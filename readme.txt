The TWW Protein Calculator Plugin for WordPress allows users to calculate their protein intake based on various parameters.

== Description ==

The TWW Protein Calculator Plugin for WordPress provides a simple, yet powerful tool for individuals looking to manage their dietary protein intake based on various factors such as weight, activity level, and conditions like pregnancy or lactation. The plugin features an easy-to-use shortcode that can be integrated into posts and pages, along with a comprehensive admin panel for custom settings.

**Features Include:**
- Dynamic protein calculation based on user input.
- Support for both metric and imperial unit systems.
- Customizable settings for activity levels and special conditions.
- Shortcode integration for easy deployment on any post or page.

== Installation ==

1. Upload `twwc-protein-calculator` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode `[twwc_protein_calculator]` in your posts or pages to display the protein calculator.

== Frequently Asked Questions ==

= How do I display the protein calculator on my site? =
Use the shortcode `[twwc_protein_calculator]` in your posts, pages, or widget areas.

= Can I customize the calculator appearance? =
Yes, you can apply custom CSS to alter the appearance of the calculator to match your site's design.

= Is the plugin compatible with all WordPress themes? =
Yes, the plugin is designed to be compatible with most themes but optimal performance is with themes that follow WordPress coding standards.
